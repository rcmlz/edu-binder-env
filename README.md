## [JavaScript](https://de.wikipedia.org/wiki/JavaScript)

[JavaScript](https://de.wikipedia.org/wiki/JavaScript) (kurz JS) ist eine Skriptsprache, die ursprünglich 1995 von Netscape für dynamisches HTML in Webbrowsern entwickelt wurde, um Benutzerinteraktionen auszuwerten, Inhalte zu verändern, nachzuladen oder zu generieren und so die Möglichkeiten von HTML und CSS zu erweitern.
